menu
directive in module ui

Description

Insert multi level menu

Usage

as element:
<menu
       links="{string}">
</menu>

Parameters

ParamTypeDetails
linksstring

JSON string to deserialize

   [
   {
     text: 'Item 1',
     url: '/item-1',
     path: '/item-1'
   },
   {
     text: 'Item 2',
     url: '/item-2',
     path: 'item-2',
     submenu: [
       {
         text: 'Item 2-1',
         url: '/item-2/item2-1',
         path: '/item-2/item2-1'
       }
     ]
   }
   ]
