# angular-cd-directive

Directive to display multi-level menu from json

***


## Requirement

* **AngularJS**: https://angularjs.org


### Development

```bash
$ npm install -g grunt-cli
$ npm install & bower install
$ grunt
```


##Testing
```bash
$ grunt test
```

##Display documentation
```bash
$ grunt showDoc
```

##Display example
```bash
$ grunt serve
```
